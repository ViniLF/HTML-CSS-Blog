A História do HTML

O HTML (HyperText Markup Language) foi criado em 1991 por Tim Berners-Lee, com o objetivo de permitir a formatação e o link entre páginas na recém-criada World Wide Web (WWW). A primeira versão do HTML era simples, permitindo apenas a criação de links e formatação básica de texto.

Com o tempo, o HTML evoluiu para suportar novas funcionalidades e elementos, como imagens, tabelas e formulários. Em 1995, o HTML 2.0 foi lançado como o primeiro padrão oficial, seguido de várias atualizações, com o HTML 4.01 em 1999, marcando uma grande evolução na estruturação de documentos.

HTML na Atualidade

Atualmente, o HTML 5 é a versão mais moderna e amplamente utilizada. Introduzido oficialmente em 2014, o HTML5 trouxe importantes inovações, como a integração de áudio e vídeo diretamente nas páginas, suporte a gráficos 2D e 3D, além de melhorias na semântica e acessibilidade.

Hoje, o HTML é uma das tecnologias fundamentais da web, usado em praticamente todos os sites, junto com CSS e JavaScript. Através do HTML5, o desenvolvimento web tornou-se mais interativo e acessível, permitindo a criação de sites e aplicações responsivas e dinâmicas.

1. Título e Subtítulo

Em HTML, você pode usar as tags <h1> a <h6> para criar títulos e subtítulos. O <h1> é o título principal e <h2> a <h6> são subtítulos com diferentes níveis de importância.


<h1>Este é um título principal</h1> /* Define um título principal de nível 1 */
<h2>Este é um subtítulo de nível 2</h2> /* Define um subtítulo de nível 2 */
<h3>Este é um subtítulo de nível 3</h3> /* Define um subtítulo de nível 3 */

2. Parágrafo Simples

Um parágrafo em HTML é definido usando a tag <p>.


<p>Este é um parágrafo de exemplo.</p> /* Cria um parágrafo com texto */

3. Link (Âncora)

Para criar um link, usamos a tag <a> com o atributo href:


<a href="https://www.example.com">Clique aqui para acessar o site</a> /* Cria um link que
direciona para o URL especificado */

4. Imagem

Imagens são adicionadas com a tag <img> e o atributo src:


<img src="imagem-exemplo.jpg" alt="Descrição da imagem"> /* Adiciona uma imagem com
fonte e descrição alternativa */

5. Lista Ordenada

Um exemplo de lista ordenada (numerada) em HTML:


<ol> /* Início de uma lista ordenada */
  <li>Item 1</li> /* Item 1 da lista */
  <li>Item 2</li> /* Item 2 da lista */
  <li>Item 3</li> /* Item 3 da lista */
</ol> /* Fim da lista ordenada */

6. Lista Não Ordenada

Uma lista não ordenada (com marcadores) em HTML:


<ul> /* Início de uma lista não ordenada */
  <li>Item A</li> /* Item A da lista */
  <li>Item B</li> /* Item B da lista */
  <li>Item C</li> /* Item C da lista */
</ul> /* Fim da lista não ordenada */

7. Tabela

Você pode criar uma tabela com a tag <table>:


<table> /* Início de uma tabela */
  <tr> /* Linha da tabela */
    <th>Nome</th> /* Cabeçalho da coluna Nome */
    <th>Idade</th> /* Cabeçalho da coluna Idade */
  </tr> /* Fim da linha da tabela */
  <tr> /* Linha da tabela */
    <td>João</td> /* Dados da coluna Nome */
    <td>30</td> /* Dados da coluna Idade */
  </tr> /* Fim da linha da tabela */
  <tr> /* Linha da tabela */
    <td>Maria</td> /* Dados da coluna Nome */
    <td>25</td> /* Dados da coluna Idade */
  </tr> /* Fim da linha da tabela */
</table> /* Fim da tabela */

8. Formulário

Um exemplo simples de formulário HTML:


<form action="/submit" method="post"> /* Início do formulário com método de envio POST */
  <label for="name">Nome:</label> /* Rótulo para o campo de nome */
  <input type="text" id="name" name="name" required> /* Campo de entrada para nome */
  
  <label for="email">E-mail:</label> /* Rótulo para o campo de e-mail */
  <input type="email" id="email" name="email" required> /* Campo de entrada para e-mail */
  
  <button type="submit">Enviar</button> /* Botão de envio do formulário */
</form> /* Fim do formulário */

A História do CSS

O CSS (Cascading Style Sheets) foi desenvolvido por Håkon Wium Lie e lançado pela primeira vez em 1996. O CSS surgiu para resolver a necessidade de separar a apresentação visual do conteúdo HTML, permitindo que a estilização das páginas web fosse feita de forma mais flexível e organizada.

As primeiras versões do CSS permitiram apenas o controle básico de estilos, como fontes e cores. Desde então, o CSS evoluiu significativamente, com o CSS2 trazendo a capacidade de criar layouts complexos e o CSS3 introduzindo novas funcionalidades, como gradientes, animações e transformações.

CSS na Atualidade

Hoje, o CSS é uma linguagem poderosa para o design web. O CSS3 é amplamente utilizado e continua a ser aprimorado com novas especificações, como o Flexbox e o Grid Layout, que facilitam a criação de layouts responsivos e complexos. O CSS também suporta variáveis, custom properties e pré-processadores, que tornam a estilização mais eficiente e dinâmica.


1. Fontes e Estilos de Texto

Definindo a fonte, tamanho e estilo de texto:

/* Definindo a fonte, tamanho e estilo de texto */
    body {
        font-family: 'Arial', sans-serif; /* Define a família de fontes como Arial ou sans-serif como fallback */
        font-size: 16px; /* Define o tamanho da fonte padrão para 16 pixels */
        color: #333; /* Define a cor do texto como cinza escuro */
        line-height: 1.5; /* Define o espaçamento entre as linhas de texto */
    }
      
      /* Estilizando títulos */
    h1, h2, h3 {
        font-weight: bold; /* Define que os títulos devem ser em negrito */
        margin-bottom: 20px; /* Adiciona um espaçamento abaixo de cada título */
    }
      
    h1 {
        font-size: 2.5rem; /* Define o tamanho da fonte do título h1 para 2.5 rem */
    }
      
    h2 {
        font-size: 2rem; /* Define o tamanho da fonte do título h2 para 2 rem */
    }
      
    p {
        font-size: 1rem; /* Define o tamanho da fonte dos parágrafos para 1 rem */
        margin-bottom: 15px; /* Adiciona um espaçamento abaixo de cada parágrafo */
    }

2. Cores e Fundos

Definindo cores de fundo e texto, e alterando cores ao passar o mouse:

/* Definindo cores de fundo e texto */
    body {
        background-color: #f4f4f4; /* Define a cor de fundo da página como cinza claro */
        color: #333; /* Define a cor do texto como cinza escuro */
    }
      
      /* Alterando cor de fundo e texto ao passar o mouse */
    button {
        background-color: #007bff; /* Define a cor de fundo do botão como azul */
          color: white; /* Define a cor do texto do botão como branco */
          border: none; /* Remove qualquer borda do botão */
          padding: 10px 20px; /* Adiciona espaçamento interno de 10px em cima e embaixo e 20px
          nas laterais */
          border-radius: 5px; /* Arredonda os cantos do botão */
          cursor: pointer; /* Mostra o ponteiro de "mão" ao passar o mouse */
    }
      
    button:hover {
        background-color: #0056b3; /* Altera a cor de fundo do botão para um azul mais escuro ao
        passar o mouse */
    }

3. Espaçamento (Margin e Padding)

Espaçamento entre elementos usando margin e padding:

/* Espaçamento entre elementos */
    .container {
        padding: 20px; /* Adiciona 20px de espaçamento interno (padding) em todos os
        lados do container */
        margin: 0 auto; /* Centraliza o container horizontalmente */
    }
      
    p {
        margin-bottom: 1.5rem; /* Adiciona um espaçamento inferior de 1.5 rem em cada parágrafo */
    }
      
    h1 {
        margin-top: 0; /* Remove o espaçamento superior do título h1 */
        margin-bottom: 1rem; /* Adiciona 1 rem de espaçamento inferior no título h1 */
    }

4. Bordas e Sombras

Bordas e sombras em caixas para melhorar a aparência:

/* Bordas e sombras em caixas */
    .box {
        border: 1px solid #ccc; /* Adiciona uma borda de 1px com cor cinza clara */
        padding: 20px; /* Adiciona 20px de espaçamento interno (padding) dentro da caixa */
        box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1); /* Adiciona uma sombra suave à caixa */
        border-radius: 8px; /* Arredonda os cantos da caixa */
    }

5. Layout Responsivo com Flexbox

Usando Flexbox para criar layouts flexíveis e responsivos:

/* Usando Flexbox para criar layout flexível */
    .container {
        display: flex; /* Ativa o layout flexível (flexbox) */
        flex-direction: row; /* Organiza os itens em linha (horizontalmente) */
        justify-content: space-between; /* Distribui os itens com espaçamento igual entre eles */
    }
      
    .container .item {
        flex: 1; /* Define que cada item flexível ocupa uma fração igual do espaço */
        padding: 10px; /* Adiciona 10px de espaçamento interno (padding) em cada item */
        margin: 5px; /* Adiciona 5px de margem ao redor de cada item */
        background-color: #e9ecef; /* Define a cor de fundo dos itens como cinza claro */
    }

6. Grid Layout

Usando Grid Layout para organizar elementos em uma grade:

/* Layout de Grid para organizar elementos */
    .grid-container {
        display: grid; /* Ativa o layout de grade (grid) */
        grid-template-columns: repeat(3, 1fr); /* Define três colunas de largura igual */
        gap: 10px; /* Adiciona 10px de espaçamento entre os itens da grade */
    }
      
    .grid-item {
        background-color: #ddd; /* Define a cor de fundo dos itens da grade como cinza */
        padding: 20px; /* Adiciona 20px de espaçamento interno (padding) nos itens */
        text-align: center; /* Centraliza o texto dentro dos itens da grade */
    }

7. Media Queries (Responsividade)

Adaptação de layout para diferentes tamanhos de tela usando Media Queries:

/* Adaptação de layout para telas menores */
        @media (max-width: 768px) {
            .container {
            flex-direction: column; /* Organiza os itens em coluna (verticalmente) em telas menores */
         }
      
            .grid-container {
            grid-template-columns: 1fr; /* Define uma única coluna em vez de três em telas menores */
         }
    }

No exemplo acima, usamos a tag <img> para exibir uma imagem e a envolvemos dentro de um contêiner chamado .image-container. O CSS foi utilizado para estilizar a imagem, aplicando arredondamento nos cantos e um efeito de zoom ao passar o mouse.

HTML:

<div class="image-container">
    <img src="images/cachoeira.jpg" alt="Imagem de Cachoeira" class="styled-image">
</div>

CSS:

.image-container {
    display: block; /* Faz com que o container se comporte como um bloco, ocupando a
    largura disponível */
    width: 50%; /* Define a largura do container como 50% da largura do elemento pai */
    padding: 20px; /* Adiciona 20px de espaço interno ao redor do conteúdo do container */
    background-color: white; /* Define a cor de fundo do container como branca */
    border-radius: 10px; /* Adiciona bordas arredondadas com um raio de 10px */
    box-shadow: 1px 4px 15px rgba(0, 0, 0, 0.5); /* Adiciona uma sombra ao redor do container,
    com deslocamento de 1px horizontal, 4px vertical e um desfoque de 15px */
    margin: 20px auto; /* Adiciona uma margem de 20px na parte superior e inferior e centraliza?
    o container horizontalmente */
}

.styled-image {
    width: 100%; /* A imagem ocupa 100% da largura do container */
    max-width: 400px; /* Define o tamanho máximo da imagem */
    border-radius: 15px; /* Arredonda os cantos da imagem */
    transition: transform 0.3s ease; /* Adiciona uma transição suave */
}

.styled-image:hover {
    transform: scale(1.05); /* Aumenta a imagem levemente ao passar o mouse */
}
